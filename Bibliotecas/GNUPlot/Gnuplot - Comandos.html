Gnuplot


Gnuplot es un programa freeware que permite realizar  grcas con datos en  2D y en 3D.  Gnuplot puede usarse en muchos ambientes computacionales, como son:  Linux, IRIX, Solaris, Mac OS X y  Windows. 


Para funcionar  requiere  mnimas capacidades en tarjetas grcas. Posee una amplia variedad de opciones para generar grcas.


La pgina oficial es http://www.gnuplot.info/


Modo de empleo 


En Linux, bastar con teclear gnuplot, para ingresar a la aplicacin. Figura1.



Gnuplot cuenta con los siguientes comandos:


 Instruccin Funcin 
 plotGrafican funciones y/o datos en dos dimensiones.
 splotGrafican funciones y/o datos en tres  dimensiones.     
 set xlabel "EJE X"Rotula el eje X
 set ylabel "EJE Y"Rotula el eje Y
 set xrangeCambia el rango del eje X
 set yrangeCambia el rango del eje Y
 set autoscaleGnuplot determina los rangos
 set labelColoca una etiqueta dentro de la grfica
 unset labelElimina las etiquetas
 set title "TITULO Coloca el titulo de la grfica    
 set gridMuestra  el fondo de la imagen con cuadricula
 set terminalIndica el tipo de salida que la grfica 
 set output

Ejemplo:


set title "Factor Gamma" 

c=3 

f(x)=1/sqrt(1-x**2/c**2) 

set xrange[0:3] 

set xlabel "Velocidad v [*10^8 m/s]" 

set ylabel "Gama" 

set yrange[0:7] 

set key title "Funcion" 

set grid 

set terminal postscript color enhanced 

set output "gamma.ps" 

plot f(x) title "1/(1-v^{2}/c^{2})^{1/2}" 

replot
